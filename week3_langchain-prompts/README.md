#  Week 3: LangChain Prompt Engineering Assignment

This project contains solutions to a series of exercises focused on prompt engineering using LangChainand Google Gemini LLM.

The goal is to understand how to build structured prompts, chain them with models, and simulate intelligent conversational systems.

---

## 🚀 Tech Stack

- Python 3.11+
- LangChain
- Google Gemini (gemini-2.5-flash)
- python-dotenv
